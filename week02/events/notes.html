    1. Local Storage:

      • Check out this site.
      • Local storage: data that persists even if you refresh the page or close the browser.
        Session storage: data that will get cleared when the browser is closed.
      • Storage capcities:
        Local Storage = 5MB
        Cookie storage = 4KB
      • Using local storage you can:
        Set key:value pairs.
        Only store strings.
        Arrays and objects can be stored but you must first convert them to string using JSON.stringify() and JSON.parse().
      • Local storage methods
        setItem(key,value) - add key & value to local storage.
        getItem(keyName) - retrieve a value by the key.
        removeItem(keyName) - remove a value by the key.
        clear() - clear all storage.
        Examples:
        localStorage.setItem('name','Pedro')
        sessionStorage.setItem('name','Side Thai')
        localStorage.getItem('name')
      • To see your local storage, open developer console and select Storage tab.
