    1. This keyword

      • A method is a function that is declared/used inside of an object. When they use this keyword, they are referencing the Object they are inside.
        let user = {
  name: "John",
  age: 30,

  sayHi() {
    // "this" is the "current object"
    alert(this.name);
  }

};

user.sayHi(); // John
      • Arrow functions have no this. When this is used inside an arrow function inside an object, the this will refer to the object of the first 'normal' function outside of the arrow function.
        let user = {
  firstName: "Ilya",
  sayHi() {
    let arrow = () => alert(this.firstName);
    arrow();
  }
};

user.sayHi(); // Ilya

    2. Objects

      • Objects are a collection of key-value pairs separated by commas, placed inside curly braces {}.
      • Objects have properties that can be any JavaScript value. If a property is a function, it’s known as a method.
      • An object’s properties and methods can be accessed using either dot notation or by square bracket notation.
      • Objects are mutable, which means their properties and methods can be changed or removed even if they were declared using CONST.
      • JSON is a portable data format that uses JavaScript object literals to exchange information.
      • The Regex object can be used to create regular expressions.
      • Object literals allow JavaScript to create objects without having to create a class first, like in other programming languages. Object literals are made by simply wrapping an object's properties and methods in curly braces.
      • ES6 provided a shorthand method of creating objects if a property key is the same as a variable name that the property value is assigned to:
        const name = 'Iron Man';
const realName = 'Tony Stark';
// long way
const ironMan = { name: name, realName: realName };
// short ES6 way
const ironMan = { name, realName };
      • To access properties, use either dot notation (superman.realName) or square bracket notation (superman['realName']).
      • To list all properties of an object use: 
        for(const key in superman) {
         console.log(key + ": " + superman[key]); 
        }
      • To list an object properties, excluding inherited properties, use:
        for(const key in superman) {
         if(superman.hasOwnProperty(key)){
console.log(key + ": " + superman[key]); 
        }
}
      • To delete a property from an object, use: 
        delete superman.fly
      • JSON rules:
        1. Property names must be double-quoted
        2. Permitted values are double-quoted strings, numbers, true, false, null, arrays and objects
        3. Functions are not permitted values
      • The parse() method takes a string of data in JSON format and returns a JavaScript object.
      • The stringify() method does the opposite, taking a JavaScript object and returning a string of JSON data.
      • The Math.random():
        1. Math.random() will generate a number between 0 (inclusive) and 1 (exclusive)
        2. To generate a random number between 0 and another number, multiply the value by that number. Ex: 6 * Math.random();
        3. Example to generate a random integer: Math.floor(6 * Math.random()); Math.trunc(20 * Math.random()); Math.ceil(100 * Math.random());
      • "The Epoch" refers to the date Jan 01, 1970 in context of the Date object.
      • UTC time stands for Universal Time Coordinated.
      • Use getFullYear() to get the year from your Date object. (Don't use the getYear() getter - it is not Y2K compliant and is thus not accurate.)
      • Regular Expression constructor has the advantage of letting you create a regular expression using a variable:
        const language = 'JavaScript';
const pattern = new RegExp(language);

    3. Document Object Model (DOM)

      • Definition: The DOM treats everything on a web page as a node. HTML tags, the text inside these tags, even the attributes of a tag are all nodes. The HTML tag is the root node, and every other part of the document is a child node of this.
      • Turn node lists into arrays:
        1. Use Array.from(): const imageArray = Array.from(document.images);
        2. Use the spread operator: const imageArray = [...document.images];
      • Common DOM methods:
        1. getElementById(): const custName = document.getElementById('customerName');
        2. getElementsByTag(): const inputNodeList = document.getElementById('input'); (returns a node list of all elements with tag.)
        3. getElementsByClassName(): const villainsNodeList = document.getElementsByClassName('villians'); (returns a node list of all elements with class.)
        4. querySelector(): const bats = document.querySelector('#bats'); (will return the first element in the document that matches the CSS selector provided, which in this example is the id="bats". Use this inplace of getElementById.)
        5. querySelectorAll(): const heroes = document.querySelectorAll('.hero'); (will return node list of all elements in the document that matche the CSS selector provided, which in this example is the class="hero". Use this inplace of getElementsByClassName.)
      • jQuery: jQuery is a popular JavaScript framework that makes it very easy to find elements on a page using a CSS-style syntax. It uses document.querySelectorAll() in the background whenever it can. For example, the jQuery code ,$('ul#roster').find('li#bats'); is basically doing the same as our previous example:
        const ul = document.querySelector('ul#roster');
ul.querySelector('li#bats')
