    2. Mobile First Video

      1. Speed matters! Reduce images on webpage designs unless necessary. If an image is needed, choose to use a SVG/Webfonts image. They are much smaller filesizes and consume much less bandwidth; they are scalable on any screen size so their quality won't be affected.
      2. Before loading webpage for consumption, MINIFY. This will remove white space from your CSS and JS code. (Research MINIFY)
      3. GZIP files to further compress the JS and CSS code so that the webpage loads faster.(Research GZIP)
      4. FLOAT LABELS???? First time seeing these. How do you make/incorporate them?
      5. SHOW PASSWORDS - make this default in your signin designs. Have HIDE option available for users who wish to conceal their password entries. Showing passwords as users type in is tested to be faster and prevent users from doing other steps like retrieving forgotten passwords or resetting passwords that have been forgotten.
      6. MAKE ALL PHONE NUMBERS TAPPABLE/CLICKABLE: Ex - <a href="tel:12223334444">1-222-333-4444</a>
      7. DON'T EVER USE POP-UPS/POP-OVERS!!!!!!
